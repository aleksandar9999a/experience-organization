# Experience Organizer


## What is this?
* Web Application.
* Project for SoftUni VueJS 2020.
* A solution for people looking for how to manage their projects.

## Features
* Authentication
* Create new projects
* Manage projects
* Track their progress
* etc. 

## Dependencies
* VueJS
* VueMatererial
* Vuelidate
* Vue Router
* Firebase

## How to run it?

```bash
    npm install

    npm run serve
```
